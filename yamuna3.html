frontend
The frontend is the part of a website or web application that users directly interact with, including everything they see, hear, or use to interact with the application, such as layout design, text, images, buttons, and navigation menus.
back end
The backend, or server-side, of a web application or software is the part that handles data management, logic, and server-side operations, everything the user doesn't directly interact with, ensuring the application functions smoothly and efficiently.
